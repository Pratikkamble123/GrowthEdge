# 🚀 GrowthEdge – Predictable Lead Generation Platform

## 🌐 Overview

**GrowthEdge** is a full-stack lead generation platform designed to help businesses scale through data-driven marketing strategies.

The platform provides a modern web interface where businesses can showcase their services, generate leads, and convert visitors into potential clients using high-conversion marketing funnels.

This project demonstrates a **modern full-stack architecture** using React for the frontend, Express for the backend, and PostgreSQL for database management.

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Pratikkamble123/GrowthEdge.git
```

Navigate to the project folder:

```
cd GrowthEdge
```

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```


## 🔑 Environment Variables

Create a `.env` file in the root directory:

```
DATABASE_URL=your_neon_postgresql_connection
PORT=5000
```
